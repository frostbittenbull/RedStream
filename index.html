RedStream

Простой и удобный десктопный загрузчик видео с минималистичным интерфейсом для YouTube, Instagram, TikTok и сотен других платформ.

Скачать для Windows (x64)

Возможности

Форматы видео и аудио

Скачивание видео в MKV и MP4, аудио в OPUS, M4A и MP3. Поддержка YouTube, Instagram, TikTok и сотен других сайтов.

Гибкие настройки качества

Динамический выбор разрешения, кодека (AV1, VP9, H.264) и FPS на основе реальных форматов видео. Умный подбор AV1 с автоматическим фолбэком.

Предпросмотр видео

Перед скачиванием программа показывает превью, название и длительность видео — чтобы вы точно знали, что загружаете.

Обрезка на лету

Обрезка видео по точным таймкодам прямо во время скачивания. Не нужно загружать файл целиком — только нужный фрагмент.

Детальный прогрессбар

Отображение процента, размера файла, типа потока (видео/аудио) и скорости загрузки. Прогресс также виден в заголовке окна.

История загрузок

Приложение сохраняет историю скачанных ссылок — легко найдёте нужное видео и повторите загрузку в другом формате.

Планировщик загрузок

Запланируйте скачивание на конкретную дату и время через встроенный календарь. Работает в фоне даже при закрытом окне интерфейса.

Уведомления Windows

По завершении загрузки приложение отправляет системное уведомление Windows — вы не пропустите готовый файл.

Системный трей

Сворачивание в системный трей для ненавязчивой работы в фоне. Настройка автозапуска при старте Windows прямо в интерфейсе.

Поддержка прокси

Протоколы HTTP, SOCKS4 и SOCKS5 с авторизацией по логину и паролю. Настройка через Дополнительно → Настроить прокси.

Авторизация через браузер

Скачивание видео с возрастными ограничениями (18+) с использованием куки вашего браузера. Просто выберите браузер в списке.

Автообновление компонентов

При каждом запуске RedStream автоматически проверяет обновления для самой программы, yt-dlp и ffmpeg. Обновление — одной кнопкой внутри приложения.

Интерфейс

Установка

Вариант 1 — Готовый установщик (рекомендуется)

1. Скачайте последний релиз из раздела Releases.
2. Запустите RedStream_Setup_x64.exe и следуйте инструкциям на экране.

Вариант 2 — Из исходников

Выполните следующие команды в терминале:

git clone https://github.com/frostbittenbull/RedStream.git
                cd RedStream
                curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe -o yt-dlp.exe
                curl -L https://www.gyan.dev/ffmpeg/builds/ffmpeg-git-essentials.7z -o ffmpeg.7z
                # Распаковать ffmpeg.7z и положить ffmpeg.exe рядом с redstream.py
                pip install -r requirements.txt
                python redstream.py
  • Зависимости: customtkinter, Pillow, winotify, pystray
  • Требуется: yt-dlp.exe и ffmpeg.exe в папке с приложением (или в PATH)

Как пользоваться

Куда сохраняются файлы

Все загруженные файлы сохраняются в папку RedStream Downloader внутри вашей стандартной папки «Загрузки». Путь можно изменить прямо в интерфейсе программы.

Обрезка видео

Нажмите кнопку ⚙ рядом с полем ввода ссылки. В открывшемся окне укажите таймкоды «Начало» и «Конец» в формате ЧЧ:ММ:СС.

Если поля оставить пустыми — видео скачается целиком.

Планировщик загрузок

Нажмите Дополнительно → Планировщик:

  1. Укажите дату и время — вручную или через встроенный календарь.
  2. Добавьте одну или несколько ссылок.
  3. Выберите формат, кодек и разрешение.
  4. Нажмите ЗАПЛАНИРОВАТЬ.

Таймер работает в фоне — окно можно закрыть. Статус восстанавливается при повторном открытии. Для отмены нажмите ОТМЕНА.

Авторизация через браузер

Для видео с возрастными ограничениями выберите ваш браузер в выпадающем списке. Программа использует сохранённые куки для доступа к контенту.

⚠️ Перед скачиванием браузер должен быть закрыт.

Если видео без ограничений — выберите «Без авторизации».

Настройка прокси

Откройте Дополнительно → Настроить прокси. Поддерживаются протоколы HTTP, SOCKS4 и SOCKS5, а также авторизация по логину и паролю.

Обновление компонентов

При каждом запуске RedStream автоматически проверяет обновления программы, yt-dlp и ffmpeg. При наличии обновления появится цветной баннер с кнопкой.

Обновить вручную: Дополнительно → Обновить компоненты.
